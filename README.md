📊 Amazon E-Commerce Sales Performance Analytics - Power BI Dashboard

An interactive Business Intelligence dashboard designed in Power BI to analyze the Amazon e-commerce product pricing trajectories, inventory distribution, and performance metrics.


🚀 Executive Summary
This report provides a data-driven overview of the Amazon sales performance dataset based on key operational performance dimensions, including analytical pipeline automation, clean database integration, revenue trends, and customer product ratings mapping.


📈 Key Performance Indicators (KPIs)
Total Revenue:** \$11.73M — High-impact overall financial volume tracking across the optimized catalog spectrum.
Total Units Sold:** 630K Units — Robust market demand and delivery performance representing diversified inventory item sales.
Database Framework:** MySQL (RDBMS) — Secure and organized storage solution executing core structural data extractions.
Data Processing Layer:** Python (Pandas) — Used programmatic algorithms to transform variable elements like `price_cleaned` and `rating_cleaned`.

🛠️ Technical Pipeline Workflow

 1️⃣ Relational Data Architecture (MySQL)
* Structured raw transaction logs, item listings, and purchase metadata into normalized SQL tables.
* Ensured structural integrity and efficient query performance for data extraction.

 2️⃣ Programmatic Data Engineering (Python)
* Cleaned missing values, dropped duplicate records, and handled null (`NaN`) fields using the **Pandas** library.
* Parsed and standardized product fields to create reliable, analytical data variables.
 3️⃣ Visual Analytics & Formatting (Power BI)
* Built a polished dynamic dashboard utilizing clean semantic card design layouts.
* Applied advanced visual rules like standardizing multi-currency parameters ($ USD metrics) and using visual filtration to remove blank boxes.
* Implemented professional visual border alignments and structural drop-shadow boundaries for corporate layout presentation guidelines.
